# Анекс Тур Нижнекамск — сайт

Статический сайт турагентства (официальный франчайзинговый офис ANEX, Нижнекамск).
Живой адрес: **https://iillyyaa1997.github.io/voyage-travel-preview/**

## Как устроено
- Чистый HTML/CSS, без конструктора. Хостинг — **GitHub Pages**, деплой — **GitHub Actions** по `git push` в `main`.
- Перед публикацией прогоняется линтер вёрстки (`ci/lint.js`): горизонтальный overflow, битые картинки, ошибки вложенности тегов. Если линтер падает — сайт НЕ деплоится.
- Стоимость хостинга: 0 ₽.

## Страницы (17)
`index.html` (главная) · `nizhnekamsk.html` (туры из Нижнекамска) · 10 направлений (`turkey/egypt/uae/thailand/maldives/vietnam/georgia/abkhazia/sochi/srilanka.html`) · `offers.html` (горящие) · `faq.html` · `about.html` · `contacts.html` · `privacy.html` · `404.html`

## Как править контент
1. Открыть нужный `.html`, поправить текст/цену/телефон.
2. `git commit` + `git push` → через ~1 мин сайт обновится сам (после прохождения линтера).
3. Локально проверить вёрстку: `cd ci && npm install && node lint.js ..`

Скрипты:
- `scripts/seo-inject.py` — идемпотентно добавляет SEO-голову (favicon, Open Graph, canonical, JSON-LD) во все страницы. Запускать после добавления новой страницы.
